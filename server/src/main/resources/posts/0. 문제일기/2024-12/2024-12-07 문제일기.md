# 풀이

0 - 회문
1 - 유사회문
2 - 그 외

문자열의 길이는 10만, 테스트케이스는 30이므로 선형으로 해결 가능
회문은 선형으로 해결 가능하지만 1개씩 삭제해보며 회문 판단하는 것은 $O(|S|^2)$이 든다.

어짜피 유사 회문이려면 딱 한군데만 다르다. 투 포인터로 그 부분을 넘기고 계속했을 때 회문이면 성공이다.
